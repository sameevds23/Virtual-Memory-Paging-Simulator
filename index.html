ℹ️ How to Use This Simulator

Address Translation: Enter logical addresses and see how they're converted to physical addresses. The simulator shows page number, offset, frame number, and whether a page fault occurred.

Page Replacement: Compare FIFO (First-In-First-Out) and LRU (Least Recently Used) algorithms. See which one performs better with fewer page faults!
